# flutter_im
项目基于腾讯即时通信API,具备实时通信、通话、朋友圈分享等功能
# 目录结构
- layout 通用页面架构
- login 登录页面 
- home 消息首页
- friend 朋友页面
- circle 朋友圈
- my 我的
- friend_info 好友信息
- chart 聊天页面
- add_friend 添加好友
- new_friend 新好友通知页面
- web_view webView 页面跳转第三方地址
- circle_details 朋友圈单行详情
- circle_send 发布朋友圈
- circle_separate 个人朋友圈
